Setup files are used to format and select objects/events from GW and AGN catalogs, which are then compared using tcabrera's lambda posterior probability distribution calculation (branched here: https://github.com/mchandarkouba/tcabrera_bbhagn). 

Plotting files are to format/visualize input data or lambda posterior for publication.

Most scripts use the gw_objects package found in the main branch.
